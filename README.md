# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to delivering actionable insights through dashboards and reports. It covers data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and visualization using business intelligence tools.

The goal of this project is to showcase practical skills in data analysis, database querying, and data storytelling.

---

## Dataset

* The dataset used in this project is structured and contains relevant business or operational data.
* It includes multiple features such as numerical and categorical variables used for analysis.
* Data preprocessing was required to handle missing values, inconsistencies, and formatting issues.

## Tools & Technologies

* **Python** – Data loading, cleaning, and EDA
* **Pandas, NumPy, Matplotlib/Seaborn** – Data manipulation and visualization
* **SQL (MySQL)** – Data querying and analysis
* **Power BI** – Dashboard creation and visualization
* **Gamma** – Presentation (PPT) creation

## Project Workflow

### 1. Data Loading

* Imported dataset into Python using Pandas
* Inspected data structure, data types, and summary statistics

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized column formats and data types

### 3. Exploratory Data Analysis (EDA)

* Identified trends, patterns, and outliers
* Created visualizations for better understanding
* Performed feature-level analysis

### 4. SQL Analysis

* Loaded cleaned data into relational databases
* Executed SQL queries for deeper insights:

  * Aggregations (SUM, AVG, COUNT)
  * Joins and filtering
  * Grouping and sorting


### 5. Dashboard Development

* Built an interactive Power BI dashboard
* Included KPIs, filters, and visual insights
* Focused on user-friendly and business-oriented design

### 6. Reporting & Presentation

* Created a detailed report summarizing findings
* Designed a presentation using Gamma
* Highlighted key insights, trends, and recommendations

## How to Run

### 1. Python Analysis

```bash
# Install required libraries
pip install pandas numpy matplotlib seaborn

# Run the analysis script
python analysis.py
```

### 2. Database Setup

* Import dataset into PostgreSQL / MySQL / SQL Server
* Run SQL scripts provided in `/sql` folder

### 3. Power BI Dashboard

* Open `.pbix` file in Power BI Desktop
* Refresh data connections if needed

### 4. Report & Presentation

* Access the report document in `/report`
* View the presentation created using Gamma
